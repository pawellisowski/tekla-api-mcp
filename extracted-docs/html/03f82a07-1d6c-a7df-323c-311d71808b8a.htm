EventsModelUnloadingSync Event

The ModelUnloadingSync event is raised synchronously on the main thread just before the model is unloaded.

Use this event only when core resources need be cleaned up before the model unloading proceeds. All UI-only related resource cleanup should be done in the async ModelUnloading event.
